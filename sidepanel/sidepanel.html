LLM API 配置

摘要仅使用云端 LLM（兼容 OpenAI 接口）；未配置时不会生成本地摘要。

正文去噪只发送本地候选正文块，不发送完整 DOM、历史笔记或完整本地数据；默认关闭。

⚠ 密钥仅保存在本地浏览器中；调用云端摘要时会作为 Authorization 发送给你配置的 API 地址，且该接口需要允许浏览器扩展跨域访问。

YouTube 字幕后端

当页面内 YouTube 字幕不可用时，调用你自建的 youtube-transcript-api 后端；只发送视频 ID、URL 和语言偏好。

⚠ 字幕后端密钥仅保存在本地；密钥只作为 Authorization 请求头发送，不会进入请求正文或模型 prompt。
